Detailed Description

A DimensionStyle defines the default appearance of dimension entities in a model. There is one DimensionStyle per model, accessible via Model::dimension_style().

This is a read-only class — all properties are getters only. DimensionStyle has no Create/Release functions; it is owned by the model.
