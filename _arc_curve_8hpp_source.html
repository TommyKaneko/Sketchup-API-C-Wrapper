38/**
39 * @brief C++ wrapper for SUArcCurveRef.
40 *
41 * An ArcCurve is a special type of Curve entity that represents an arc or
42 * full circle. It is defined by a center point, start point, end point, and
43 * a normal vector. If the start and end points are the same, a full circle
44 * is generated.
45 *
46 * ArcCurve extends Entity (via SUArcCurveToEntity/SUArcCurveFromEntity).
47 *
48 * @note When released, the arc curve and its associated edge objects are
49 * also released.
50 *
51 * @see SUArcCurveRef
52 * @since SketchUp 2016, API 4.0
53 */
