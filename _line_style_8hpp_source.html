41/**
42 * @brief C++ wrapper for SULineStyleRef.
43 *
44 * A LineStyle defines the visual appearance of lines (stipple pattern, width,
45 * color, etc.). LineStyles are obtained from the model's LineStyles manager.
46 * Built-in styles cannot be modified; only user-created copies can be
47 * changed.
48 *
49 * @note There is no SULineStyleCreate function — new styles can only be
50 * created by copying an existing one via create_copy().
51 *
52 * @see SULineStyleRef
53 * @since SketchUp 2019, API 7.0
54 */
