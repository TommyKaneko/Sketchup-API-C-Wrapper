Detailed Description

MeshHelper is a helper class that tessellates a Face into triangles and provides access to the resulting vertices, normals, vertex indices, and STQ texture coordinates.

MeshHelper is a standalone helper object — it does not inherit from Entity and is not part of the model hierarchy. It must be explicitly created and released.
