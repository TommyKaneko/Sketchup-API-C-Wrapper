43/**
44 * @brief Wrapper for SUInstancePathRef, representing a path through the
45 * component hierarchy to a specific entity instance.
46 *
47 * An InstancePath describes the nesting of component instances
48 * from the model root down to a specific entity (the "leaf").
49 * It is used to resolve transformations and persistent IDs
50 * for nested geometry.
51 *
52 * @see https://extensions.sketchup.com/developer_center/sketchup_c_api/
53 */

183 /**
