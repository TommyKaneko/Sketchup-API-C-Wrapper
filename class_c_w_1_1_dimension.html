Detailed Description

A Dimension is the base class for linear and radial dimension entities. Dimensions are drawing elements that display measurement annotations in the model. This class provides access to the properties common to all dimension types (text, plane, 3D flag, arrow type, font).

Note
The SketchUp C API does not provide SUEntitiesAddDimension(). Dimensions can be created as DimensionLinear or DimensionRadial and retrieved from entities, but there is no general add-dimension method.
