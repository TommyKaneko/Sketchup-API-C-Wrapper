37/**
38 * @brief C++ wrapper for SUDimensionLinearRef.
39 *
40 * A DimensionLinear represents a linear (distance) measurement annotation
41 * in the SketchUp model. Linear dimensions measure the distance between
42 * two connection points.
43 *
44 * Linear dimensions extend the Dimension base class and inherit its common
45 * properties (text, plane, 3D flag, arrow type, font).
46 *
47 * @see SUDimensionLinearRef
48 * @since SketchUp 2017, API 5.0
49 */
