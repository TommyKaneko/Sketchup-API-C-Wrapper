Detailed Description

LoopInput holds edge indices and per-edge properties (hidden, soft, smooth, material, layer) and is consumed by Face or GeometryInput. Methods that set edge properties return a reference to this object for chaining.

See also
https://extensions.sketchup.com/developer_center/sketchup_c_api/

Adds a vertex index to the loop. LoopInput does not hold the point locations of vertices. It holds information on the edges that form the loop.

Parameters
index- the vertex index of the loop. When used with SUCreateFace() or CW::Entities, it would be a zero-based index. When used with SUGeometryInputAddFace() or CW::GeometryInput::add_face, the vertex index would need to correspond to specific indices of the vertices that have been added to SUGeometryInputRef or CW::GeometryInput object.
