Detailed Description

A DimensionLinear represents a linear (distance) measurement annotation in the SketchUp model. Linear dimensions measure the distance between two connection points.

Linear dimensions extend the Dimension base class and inherit its common properties (text, plane, 3D flag, arrow type, font).

See also
SUDimensionLinearRef

SketchUp 2017, API 5.0
