79 * @since SketchUp 2016, API 4.0

81 * @param end - the guide line end position.

85 /**
86 * @brief Creates an infinite guide line object. The generated line will be

105
106 /**
107 * @brief Retrieves the data defining the line (a point, a direction vector,
108 * and a boolean flagging if the line is infinite). For finite lines, start
109 * is the start point, and the end point can be obtained by adding the
110 * direction vector to the start point. For infinite lines, start is simply
111 * a point on the guide line, and direction is always a unit vector.
112 * @since SketchUp 2016, API 4.0
113 * @param[out] start - a point on the guide line.
114 * @param[out] direction - the guide line direction.
115 * @return true if the line is infinite, false if finite.
116 */

118
119 /**

121 * @since SketchUp 2016, API 4.0
122 */
123 bool is_infinite() const;
124
125 /**
126 * @brief Retrieves the start point of the guide line. For infinite lines,
127 * this is simply a point that lies on the line.
128 * @since SketchUp 2016, API 4.0
129 */
130 Point3D start() const;
131
132 /**
133 * @brief Retrieves the direction vector of the guide line. For infinite
134 * lines this is always a unit vector. For finite lines this is the vector
135 * from start to end.
136 * @since SketchUp 2016, API 4.0
137 */
138 Vector3D direction() const;
139
140 /**
141 * @brief For a finite guide line, returns the end point (start + direction).
142 * For infinite lines, the result is undefined.
143 * @since SketchUp 2016, API 4.0
144 */
145 Point3D end() const;
146
147 /**
148 * @brief Creates a copy of this guide line object, duplicating all properties.
149 * @return a new unattached GuideLine object with the same properties.
150 * @throws std::logic_error if this guide line is null.
151 */
152 GuideLine copy() const;
153};
