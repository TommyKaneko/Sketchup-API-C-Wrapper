Detailed Description

An EdgeUse represents the use of an edge in a specific loop/face context. It provides topology information for polygon boundaries. EdgeUse objects are obtained from loops and faces and are always read-only (no Create/Release).

The start/end vertices of an EdgeUse may differ from the underlying edge's vertices because the loop may traverse the edge in reverse.
