Detailed Description

An ArcCurve is a special type of Curve entity that represents an arc or full circle. It is defined by a center point, start point, end point, and a normal vector. If the start and end points are the same, a full circle is generated.

ArcCurve extends Entity (via SUArcCurveToEntity/SUArcCurveFromEntity).

Note
When released, the arc curve and its associated edge objects are also released.
