Detailed Description

A LineStyle defines the visual appearance of lines (stipple pattern, width, color, etc.). LineStyles are obtained from the model's LineStyles manager. Built-in styles cannot be modified; only user-created copies can be changed.

Note
There is no SULineStyleCreate function — new styles can only be created by copying an existing one via create_copy().
See also
SULineStyleRef

SketchUp 2019, API 7.0
