Detailed Description

A group is a special type of component instance that is always unique. Groups inherit from ComponentInstance and provide direct access to their entities, name, and transformation.

Note
Groups must always be attached to a parent. There is no release function for groups in the C API.
See also
SUGroupRef in the SketchUp C API
